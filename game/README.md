# Carpeta del juego

Esta carpeta contiene lo propio del juego Minecraft Live Arena.

- `mod-src/`: codigo fuente del mod Forge 1.20.1.
- `world-template/`: reservado para plantillas de mundo si se entregan mas adelante.

El mod compilado entregable se deja en:

```txt
dist/mods/minecraft_live_arena-*.jar
mods/minecraft_live_arena-*.jar
```

No subir caches de Gradle, builds temporales ni mundos generados.
