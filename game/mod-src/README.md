# Minecraft Live Arena Mod

Mod Forge 1.20.1 del prototipo de arena.

Responsabilidad:

- leer `server/config/minecraft_live_arena/live_events.jsonl`;
- reconstruir la arena circular;
- crear avatares por viewer;
- congelarlos durante inscripciones;
- liberarlos al iniciar pelea;
- aplicar acciones simples como curar/equipar.

Este mod es el punto que faltaba para que Minecraft haga cosas dentro del mundo.
